Cara Kerja Automation Content Creation

Dari satu topik → caption, referensi gambar, format, mockup brief, dan Google Doc otomatis.

1. Fondasi: Framework WAT

Sistem dibagi 3 lapisan supaya andal — AI mengurus "berpikir", kode mengurus "eksekusi".

Workflow

Instruksi

SOP dalam Markdown yang menjelaskan langkah & aturan. File: workflows/content_creation.md

Agent

Pengambil Keputusan

Claude membaca workflow, memanggil tools dengan urutan benar, menangani error.

Tools

Eksekusi

Script Python di tools/ yang benar-benar bekerja: panggil API, olah data.

2. Alur Kerja (Pipeline)

Inilah yang terjadi setiap kali kamu menjalankan satu topik:

Input

Kamu ketik 1 topik

python3 run.py "Gravestone Doji candle"

⚡ Berjalan PARALEL (3 sekaligus — hemat waktu)
Langkah 1a

Buat Caption

generate_captions.py

Claude bikin caption Instagram, Threads, X (English).

✓ JALAN
Langkah 1b

Cari Gambar Ref

search_image_refs.py

Claude susun query → BrowserAct scrape Bing Images → URL gambar asli.

✓ JALAN
Langkah 1c

Tentukan Format

determine_format.py

Claude putuskan: single image atau carousel?

✓ JALAN
Langkah 2

Buat Mockup Brief

generate_mockup_brief.py

Claude tulis brief per slide: headline, body, CTA, catatan visual. (butuh hasil format dari langkah 1c)

✓ JALAN
Langkah 3

Compile ke Google Docs

compile_to_gdocs.py

Semua hasil disusun jadi 1 Google Doc rapi dengan 3 bagian: Caption, Referensi Gambar, Mockup Brief.

✓ JALAN
Output

Link Google Doc siap pakai

Tim desain tinggal buka, eksekusi, sesuaikan brand guideline.
